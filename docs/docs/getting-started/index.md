# Getting Started

Get up and running with DaoFlow in under 5 minutes.

## Prerequisites

- Docker Engine 20.10+
- Docker Compose v2+
- Node.js 18+ or Bun 1.0+
- PostgreSQL 17

## Quick Install

```bash
# Clone the repository
git clone https://github.com/daoflow/daoflow.git
cd daoflow

# Install dependencies
bun install

# Start the dev environment
docker compose up -d
bun run dev
```

Continue to [Installation](/docs/getting-started/installation) for detailed setup instructions.
